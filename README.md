# ⚔️ RiKitPvP

Um plugin básico de **KitPvP** para servidores Minecraft.  
Este projeto foi criado apenas para fins de **estudo e demonstração**.

---

## 📌 Sobre o projeto

O **RiKitPvP** é um plugin simples que simula um sistema de KitPvP, incluindo funcionalidades básicas como kits e combate entre jogadores.

> ⚠️ **Atenção:** Este projeto não está finalizado.

---

## 🚫 Aviso importante

❗ **O código está quebrado e não deve ser utilizado em servidores reais.**  
Este repositório foi criado **apenas para demonstração de código** e testes.

- Pode conter bugs  
- Pode não funcionar corretamente  
- Não foi otimizado  
- Não é seguro para produção  

---

## 🎯 Objetivo

O objetivo deste projeto é:

- Servir como exemplo de estrutura de plugin  
- Ajudar no aprendizado de desenvolvimento com Spigot/Bukkit  
- Testar ideias relacionadas a KitPvP  

---

## 🛠️ Tecnologias utilizadas

- Java  
- API Spigot/Bukkit
